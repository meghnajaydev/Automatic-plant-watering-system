# Automatic-plant-watering-system
Built an IoT-based automatic plant watering system using soil moisture sensors to monitor real-time soil conditions and control irrigation automatically. The system reduces water wastage, eliminates manual watering, and enables remote monitoring through an IoT platform.


Components Used
Arduino UNO
Rain Sensor
Water Level Sensor
Relay Module
Water Pump
Buzzer

Working
Detects rainfall
Activates pump to store water
Automatically stops when tank is full

Applications
Smart agriculture
Rainwater conservation
IoT-based water management
